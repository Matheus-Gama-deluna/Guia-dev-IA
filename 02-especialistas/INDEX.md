# Índice de Especialistas

Este diretório contém **personas de IA** especializadas para cada etapa do desenvolvimento.

## Como Usar

1. Copie o conteúdo do especialista relevante
2. Cole como contexto/system prompt na sua IA
3. Interaja usando os prompts sugeridos
